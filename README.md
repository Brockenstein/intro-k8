# intro-k8-dev-nexus
Introduction to Kubernetes for 2018 Dev Nexus


Install Hugo

https://gohugo.io/getting-started/installing/

Mac
```
brew install hugo
```

Windows
```
choco install hugo -confirm
```


Running Hugo Site
```
cd introk8
hugo server
```

Navigate to 

[http://localhost:1313](http://localhost:1313)
